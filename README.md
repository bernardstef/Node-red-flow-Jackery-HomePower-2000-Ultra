Using great work from https://github.com/iLLixM/jackery_home_cloud-ha, help from Copilot,

I created this flow to get data , status, and state of charge from Jackery HomePower 2000 Ultra

This flow is ready to import into node red. No dependancies needed. Add your credentials in the node : "Prepare login"


I may not be able to provide too much support, but I hope it won't be needed. Very simple to set-up. There is a function at the end if you have a solar router, which send the remaining power to a water heater. Might not be needed for you.
